✨ 幻の銀水晶 - Gemini API設定ガイド ✨

Google Gemini APIを使って、AI星読み機能を無料で使えるようにしましょう！

🌙 Gemini APIとは？

Googleが提供する最新のAI APIです。無料枠があり、個人利用には十分な量が使えます。

1 Google AI Studioにアクセス

以下のリンクからGoogle AI Studioにアクセスしてください：

https://makersuite.google.com/

※ Googleアカウントでログインが必要です

2 APIキーを取得

1. 「Get API key」ボタンをクリック

2. 「Create API key」を選択

3. プロジェクトを選択（新規作成でもOK）

4. 生成されたAPIキーをコピー

⚠️ 重要： APIキーは他人に教えないでください。GitHubにアップロードしないよう注意！
3 app.htmlにAPIキーを設定

1. app.htmlファイルをテキストエディタで開く

2. 以下の行を見つける：

const GEMINI_API_KEY = 'YOUR_GEMINI_API_KEY'; // ここにAPIキーを入れる

3. 'YOUR_GEMINI_API_KEY'の部分を、取得したAPIキーに置き換える：

const GEMINI_API_KEY = 'AIzaSy...あなたのAPIキー...'; // 実際のキーに置き換え
4 ファイルを保存して使用開始

1. ファイルを保存

2. ブラウザでapp.htmlを開く

3. AI機能が使えるようになります！

🌟 使える機能

  • 📊 AI星読み分析 - あなたの星座に基づいた詳細な運勢
  • 💭 マジカルノートのAI問いかけ - 月相に応じた深い質問
  • ✨ 今日のメッセージ生成 - 毎日違うパーソナルメッセージ

💰 無料枠について

Gemini APIの無料枠：

  • 1分間に60リクエストまで
  • 1日に1,500リクエストまで
  • 個人利用には十分な量です！
✅ 設定が完了したら、app.htmlを開いて幻の銀水晶の魔法を体験してください！

🔧 トラブルシューティング

  • APIキーエラーが出る場合：キーが正しくコピーされているか確認
  • CORS エラーが出る場合：ローカルサーバーで実行するか、GitHub Pagesにアップロード
  • レート制限エラー：1分待ってから再試行
